How to Install the Custom Map

Just 3 simple steps to get your custom map running in Brawlhalla.

1
Check Your ZIP File
Every map has at least 3 files inside the ZIP — the Background, the Platform parts and a Thumbnail. Some maps have multiple platform parts.
2
Open the Brawlhalla Files
Right-click Brawlhalla in your Steam Library → Manage → Browse local files. Then navigate to the mapArt, Thumbnail and Background files.
⚠️ Tip: Make a backup copy of the original files first!
3
Replace & Enjoy
Replace the original files with my map files (keep the exact same filenames). Start Brawlhalla and enjoy my map! 🎉
📁 Where to find the map folder?
Usually located at: C:\Program Files (x86)\Steam\steamapps\common\Brawlhalla\
Look inside the mapArt subfolder.
