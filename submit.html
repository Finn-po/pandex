Creator Submission

Submit Original Game Content

Pandex accepts original, rights-cleared map backgrounds and game-related fan content for manual review. The current launch phase focuses on image previews first; playable downloads can be added later after a stricter review.

What Pandex currently accepts

Accepted

Original map backgrounds, arena concepts, preview images, creator logos and creator descriptions made by you or fully licensed to you.

Launch Focus

Brawlhalla-style map background previews are the first content category. Other games can be added later if the legal and content requirements are clear.

Submission checklist

  • Creator name or artist name
  • Map title
  • Game name the content is intended for
  • 1–4 preview images as PNG or JPG
  • Short description of the map concept
  • Optional: TikTok, Discord, YouTube, Instagram or portfolio link
  • Confirmation that you created the submitted content yourself or have the rights to publish it

Content that cannot be accepted

Protected IP

No anime characters, movie characters, brand logos, copied fan art, third-party mascots or recognizable protected characters.

Ripped assets

No extracted game files, copied textures, official artwork, modified screenshots sold as original work or stolen creator content.

Unsafe files

No executables, suspicious scripts, hidden installers or files that create security risks for users.

Unclear rights

No content where ownership, license, AI asset rights or creator permission cannot be explained clearly.

Email template

Send submissions to kontakt.pandapages@gmail.com with the subject Map Submission – Pandex.

Subject: Map Submission – Pandex Creator name: Map title: Game: Short description: Social links: Files attached: Rights confirmation: I confirm that I created the submitted content myself or have the necessary rights to publish it on Pandex. I understand that Pandex can reject, edit, hide or remove content if there are legal, quality, security or platform-policy concerns.

Review process

  • Pandex checks whether the submission fits the platform direction and content rules.
  • Approved content can first go live as preview images.
  • Downloads are only added later when file structure, rights and safety checks are clear.
  • Pandex can remove content after publication if rights issues, takedown requests or quality problems appear.
