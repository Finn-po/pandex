# Favicon Assets

Add the generated favicon files to this folder:

- `favicon-16x16.png` — 16x16 PNG
- `favicon-32x32.png` — 32x32 PNG
- `favicon-48x48.png` — 48x48 PNG
- `android-chrome-192x192.png` — 192x192 PNG
- `android-chrome-512x512.png` — 512x512 PNG
- `apple-touch-icon.png` — 180x180 PNG
- `safari-pinned-tab.svg` — monochrome SVG mask icon

Do not commit placeholder binary images. Generate these from the final PANDEX icon.
